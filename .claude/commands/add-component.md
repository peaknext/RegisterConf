description: Add a new shadcn/ui component
---

Add the shadcn/ui component: $ARGUMENTS

Steps:
1. Run: `npx shadcn@latest add $ARGUMENTS`
2. The component will be added to `src/components/ui/`
3. Import and use in your components

Example usage:
- `/add-component button` - Add button component
- `/add-component dialog` - Add dialog component
- `/add-component table` - Add table component

Available components: https://ui.shadcn.com/docs/components
